# Execute Localmente

Clonar o Projeto

```bash
git clone https://github.com/POO-2024-1/Projeto-Final.git
```

Navegar até o Diretório do Projeto

```bash
cd SCP
```

Instalar Dependências

```bash
npm install
```

Iniciar o Servidor

```bash
npm start
```
